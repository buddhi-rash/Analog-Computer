# Analog Multiplier Analog Computer Project

## Description
This project demonstrates the implementation of an analog computer using analog circuits such as adders, subtractors, and multipliers.

## Tools Used
- LTspice
- Altium Designer
- SolidWorks

## Project Contents
- Analog adder circuits
- Analog subtractor circuits
- Gilbert cell multiplier
- Logarithmic multiplier
- Differentiator
- Integrater
- LTspice simulations

## Folder Structure
Enclosure Design - Solidworks and Fusion enclosure files
Project Realization - Project outcome and demonstrations
Scehmatics and PCB - Altium schematics and PCB designs
Spice Simulations – LTspice simulation files
